## 🔑 MOLIT API 키 발급 방법

1. https://www.data.go.kr/ 접속
2. 회원가입 & 로그인
3. "아파트매매 실거래자료" 검색
4. 활용신청 → 즉시 승인
5. 마이페이지 > 인증키 복사
6. AptScore 설정에 붙여넣기
