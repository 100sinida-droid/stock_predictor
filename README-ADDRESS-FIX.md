### 3️⃣ 역거리 정확도 개선

#### 풍무서해그랑블 (양도마을)

**문제**:
```
표시: 가장가까운역: - (없음)
      역거리: 10.0km
실제: 풍무역이 1.1km에 있음
```

**원인**: DB의 역거리가 600m로 잘못 입력됨

**해결**:
```javascript
{
  n:'풍무서해그랑블',
  st:'풍무역',
  sd:1100,  // 600 → 1100 (1.1km)
}
```

**결과**:
- ✅ 가장가까운역: **풍무역**
- ✅ 역거리: **1.1km**
- ✅ 대중교통기준: **버스 환승 필요** (정확한 평가)

### 4️⃣ 점수 정확도 영향

#### 주소 수정으로 인한 점수 변화

**헬리오시티 (송파구로 수정)**:
```
변경 전: 강동구 → 지역선호도 85점
변경 후: 송파구 → 지역선호도 95점 (+10점!)
```

**이유**: 
- 강동구: 85점
- **송파구: 95점** (강남/서초 다음으로 높은 선호도)

**풍무서해그랑블 (역거리 수정)**:
```
변경 전: sd:600 → 교통 점수 과대평가
변경 후: sd:1100 → 교통 점수 정확
```

## 📝 기술 세부사항

### 주소 데이터 구조:
```javascript
{
  k: ['검색키워드1', '검색키워드2'],  // 검색용
  n: '정식명칭',                      // 표시용
  city: '서울',                       // 시/도
  gu: '송파구',                       // 구/군
  dong: '가락동',                     // 동/읍
  a: '서울 송파구 가락동 648번지',    // 전체 주소 (표시용)
  la: 37.4991,                        // 위도
  lo: 127.1219,                       // 경도
  yr: 2018,                           // 건축년도
  st: '둔촌동역',                     // 가장 가까운 역
  sd: 150                             // 역까지 거리 (미터)
}
```

### 점수 계산에 미치는 영향:

1. **gu (구)** → 지역선호도 점수
   - LOCATION_PREFERENCE[gu] 참조
   - 송파구: 95점
   - 강동구: 85점

2. **sd (역거리)** → 교통 점수
   - 150m: 100점
   - 600m: 85점
   - 1100m: 63점

3. **a (주소)** → 사용자 정보 표시
   - 지번 포함 시 더 상세
